## Why Meetily?

<Note>
**Privacy-First by Design** — While cloud meeting tools promise convenience, they create significant privacy and compliance risks. Meetily solves this with complete data sovereignty on your infrastructure, zero vendor lock-in, and full control over your sensitive conversations.
</Note>

<CardGroup cols={2}>
  <Card title="No Cloud Dependencies" icon="cloud-slash">
    Works 100% offline. No API keys required for core features.
  </Card>
  <Card title="Cost-Effective" icon="dollar-sign">
    Uses open-source AI models instead of expensive cloud APIs.

  <Card title="GDPR Compliant" icon="scale-balanced">
