## Overview

The Transcription Commands API provides access to local Whisper models for speech-to-text processing. These commands handle model discovery, downloading, loading, and transcription operations with GPU acceleration support.

## Commands

### whisper_init

Initializes the Whisper engine. Should be called once during app startup.

**Returns**: `Promise<void>`

**Rust Signature**:
```rust
#[tauri::command]
pub async fn whisper_init() -> Result<(), String>
```

**TypeScript Example**:
```typescript
import { invoke } from '@tauri-apps/api/core';

await invoke('whisper_init');
```

---

### whisper_get_available_models

Retrieves a list of all Whisper models, including their download status and specifications.

**Returns**: `Promise<ModelInfo[]>`

**TypeScript Interface**:
```typescript
interface ModelInfo {
  name: string;
  path: string;
  size_mb: number;
  status: 'Available' | 'Missing' | 'Downloading' | 'Corrupted';
  accuracy: string; // e.g., "Good", "Better", "Best"
  speed: string;    // e.g., "Fast", "Medium", "Slow"
  description: string;
}
```

**Rust Signature**:
```rust
#[tauri::command]
pub async fn whisper_get_available_models() -> Result<Vec<ModelInfo>, String>
```

**TypeScript Example**:
```typescript
import { invoke } from '@tauri-apps/api/core';

const models = await invoke<ModelInfo[]>('whisper_get_available_models');

const downloaded = models.filter(m => m.status === 'Available');
const missing = models.filter(m => m.status === 'Missing');

console.log(`Downloaded: ${downloaded.length}, Available to download: ${missing.length}`);
```

**Available Models**:
- `tiny`, `tiny.en` - Fastest, lower accuracy (~75MB)
- `base`, `base.en` - Fast, good accuracy (~150MB)
- `small`, `small.en` - Balanced speed/accuracy (~500MB)
- `medium`, `medium.en` - High accuracy, slower (~1.5GB)
- `large-v1`, `large-v2`, `large-v3` - Best accuracy, slowest (~3GB)
- `large-v3-turbo` - Optimized large model (~1.6GB)

### whisper_download_model

Downloads a Whisper model from HuggingFace. Emits progress events during download.

<ParamField path="model_name" type="string" required>
  Name of the model to download (e.g., "base", "small", "medium").
</ParamField>

**Returns**: `Promise<void>`

**Rust Signature**:
```rust
#[tauri::command]
pub async fn whisper_download_model(
    app_handle: tauri::AppHandle,
    model_name: String,
) -> Result<(), String>
```

**TypeScript Example**:
```typescript
import { invoke, listen } from '@tauri-apps/api';

// Listen for download progress
const progressUnlisten = await listen('model-download-progress', (event) => {
  const { modelName, progress } = event.payload;
  console.log(`Downloading ${modelName}: ${progress}%`);
});

const completeUnlisten = await listen('model-download-complete', (event) => {
  console.log('Download complete:', event.payload.modelName);
});

// Start download
await invoke('whisper_download_model', {
  app_handle: null, // Tauri automatically provides this
  model_name: 'base'
});

// Clean up listeners
progressUnlisten();
completeUnlisten();
```

**Events Emitted**:
- `model-download-progress`: `{ modelName: string, progress: number }`
- `model-download-complete`: `{ modelName: string }`
- `model-download-error`: `{ modelName: string, error: string }`

### whisper_load_model

Loads a downloaded Whisper model into memory for transcription. Automatically detects GPU capabilities.

<ParamField path="model_name" type="string" required>
  Name of the model to load.
</ParamField>

**Returns**: `Promise<void>`

**Rust Signature**:
```rust
#[tauri::command]
pub async fn whisper_load_model(
    app_handle: tauri::AppHandle,
    model_name: String
) -> Result<(), String>
```

**TypeScript Example**:
```typescript
import { invoke, listen } from '@tauri-apps/api';

// Listen for loading events
await listen('model-loading-started', (event) => {
  console.log('Loading model:', event.payload.modelName);
});

await listen('model-loading-completed', (event) => {
  console.log('Model loaded:', event.payload.modelName);
});

// Load model
await invoke('whisper_load_model', {
  app_handle: null,
  model_name: 'base'
});
```

**GPU Acceleration**:
- **macOS**: Automatically uses Metal + CoreML
- **Windows/Linux**: Detects CUDA (NVIDIA) or Vulkan (AMD/Intel)
- **Fallback**: Uses CPU if no GPU available

**Events Emitted**:
- `model-loading-started`: `{ modelName: string }`
- `model-loading-completed`: `{ modelName: string }`
- `model-loading-failed`: `{ modelName: string, error: string }`

### whisper_transcribe_audio

Transcribes audio data using the currently loaded Whisper model.

<ParamField path="audio_data" type="number[]" required>
  Audio samples as 32-bit floating point values (normalized to -1.0 to 1.0). Must be 16kHz sample rate.
</ParamField>

**Returns**: `Promise<string>` - Transcribed text

**Rust Signature**:
```rust
#[tauri::command]
pub async fn whisper_transcribe_audio(audio_data: Vec<f32>) -> Result<String, String>
```

**TypeScript Example**:
```typescript
import { invoke } from '@tauri-apps/api/core';

// Assume audioSamples is a Float32Array from audio processing
const audioData = Array.from(audioSamples);

const transcript = await invoke<string>('whisper_transcribe_audio', {
  audio_data: audioData
});

console.log('Transcription:', transcript);
```

**Note**: This command is typically called internally by the recording system. For live transcription, use the recording commands which handle audio processing automatically.

## Transcription Events

### transcript-update

Emitted during recording when new speech is transcribed.

**TypeScript Interface**:
```typescript
interface TranscriptUpdate {
  text: string;
  timestamp: string;
  audio_start_time: number;
  audio_end_time: number;
  duration: number;
  confidence: number;
  sequence_id: number;
}
```

**TypeScript Example**:
```typescript
import { listen } from '@tauri-apps/api/event';

const unlisten = await listen<TranscriptUpdate>('transcript-update', (event) => {
  const { text, timestamp, confidence } = event.payload;
  console.log(`[${timestamp}] ${text} (${(confidence * 100).toFixed(1)}%)`);
});
```

### transcription-error

Emitted when transcription encounters an error.

**TypeScript Example**:
```typescript
import { listen } from '@tauri-apps/api/event';

const unlisten = await listen('transcription-error', (event) => {
  const { error, userMessage, actionable } = event.payload;
  if (actionable) {
    // Show modal or prompt user action
  } else {
    // Show toast notification
  }
  console.error('Transcription error:', error);
});
```

## Model Selection Guide

### For Development
- **Recommended**: `base` or `small`
- Fast iteration and testing
- Good accuracy for most use cases

### For Production
- **Recommended**: `medium` or `large-v3`
- Best accuracy for professional transcriptions
- `large-v3-turbo` offers a good balance

### For Real-time Transcription
- **Recommended**: `tiny` or `base`
- Low latency for live captions
- Consider GPU acceleration for `small` or `medium`

### GPU Performance
- **Metal (macOS)**: 5-10x faster than CPU
- **CUDA (NVIDIA)**: 8-15x faster than CPU  
- **Vulkan (AMD/Intel)**: 3-7x faster than CPU

## Language Support

### Multilingual Models
- All non-`.en` models support 99 languages
- Automatic language detection
- Supports translation to English

### English-Only Models
- Models ending in `.en` (e.g., `base.en`, `small.en`)
- Faster processing for English audio
- Recommended for English-only use cases

**TypeScript Example**:
```typescript
import { invoke } from '@tauri-apps/api/core';

// Set language preference (optional)
await invoke('set_language_preference', {
  language: 'auto-translate' // Auto-detect and translate to English
});

// Or specify a language code
await invoke('set_language_preference', {
  language: 'es' // Spanish
});
```
